  • Looks beautiful
  • takes contact information from the user and stores it in mongodb database
  • developed using Pug, CSS and javascript on frontend
  • backend is handled in express - node.js

This is a text Utility Website, the site takes an input text from the user and then the user has to click on the operations that he wants to perform on the text from the available options, the site allows for performing multiple operations simultaneously on the text, after taking all the parameters the modifying text is shown to user.

  • Fully Responsive
  • Looks beautiful
  • The site has the ability to handle very big texts
  • used bootstrap for frontend
  • backend is handled in python - django
